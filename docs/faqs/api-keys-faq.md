# API Keys — Frequently Asked Questions

## How do I generate a new API key?

Go to Settings > API Keys > Generate New Key. Give it a descriptive name
like "production-app" or "dev-testing". Your key will be shown once —
copy it immediately and store it securely.

## How do I reset or regenerate my API key?

You cannot edit an existing key. To reset it:
1. Go to Settings > API Keys
2. Click "Revoke" next to the key you want to replace
3. Generate a new key using the steps above
4. Update your application with the new key value

## How many API keys can I have?

Free accounts: up to 3 keys. Pro accounts: unlimited keys.
Enterprise accounts: unlimited keys with optional expiry dates.

## What should I do if my API key is leaked?

Revoke it immediately from Settings > API Keys. Then audit your recent
API usage logs for unauthorized calls. Generate a new key and rotate it
in all your applications.
